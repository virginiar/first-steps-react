# First Steps React

Aplicación web basada en el proyecto first-steps del curso de "React: De cero a experto" de DevTalles en Udemy.

Se trata de un proyecto React creado utilizando Vite con TypeScript + SWC.

## Instalación y ejecución del proyecto
Descargar las dependencias con `npm install`.
Iniciar la aplicación con `npm run dev`.

## Aspectos analizados
En este proyecto se abordan los fundamentos de React:
- Componentes
- Estructura y nomenclatura
- Hook useState
- CSS condicional
- Módulos de CSS
- Impresiones de variables
- Props
